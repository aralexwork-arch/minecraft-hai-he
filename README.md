# hebrew

Hebrew localization fork of [ReWrite-Media/hai2025-ts](https://github.com/ReWrite-Media/hai2025-ts) — the MakeCode extension and tutorials for the Hour of AI 2025 Minecraft Education world.

This repository provides:
- Hebrew block translations under `_locales/he/hai2025-ts-strings.json` (and `he-IL` mirror)
- Hebrew tutorial markdown files under `_locales/he/`
- Original English fallback markdown for compatibility

Tutorials are loaded by the Minecraft Education Hebrew port via:

```
https://minecraft.makecode.com/hoc2025?ipc=1&inGame=1&noRunOnX=1&forcelang=he#tutorial:https://github.com/aralexwork-arch/hebrew/blocks_wood1
```
